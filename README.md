# Elegant Login Form

A beautiful, modern login form built with HTML, CSS, and Bootstrap 5. Perfect for your first GitHub project!

## 🚀 Features

- ✅ Modern & Elegant Design – Purple gradient with hover effects
- ✅ Fully Responsive – Displays perfectly on mobile, tablet, and desktop
- ✅ Bootstrap 5 Used – Fast and clean structure
- ✅ Font Awesome Icons – Enhanced visual appeal
- ✅ Smooth Animations – Hover effects on cards and buttons
- ✅ Validatable Form – Email and password fields included

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome Icons
